## Full description (4000 chars max)
DevPocket is the ultimate offline-first utility suite and code sandbox designed specifically for developers, engineers, and power users. Bring your workspace, testing suites, and playground tools directly onto your mobile device with zero external dependencies and 100% local execution.

Key Features:
• Code Workspace: A fully-featured code editor with syntax highlighting for JavaScript, JSON, CSS, HTML, SQL, and Markdown. Features auto-pairing brackets, adjustable font sizes, and an auto-hiding file drawer while typing.
• Brutal Formatter: Instantly beautify, prettify, or minify JSON, XML, HTML, CSS, and SQL data structure streams with offline parsers.
• Regex Playground: Design, validate, and debug regular expressions. Instantly test patterns against custom inputs with global, case-insensitive, multiline, and dotAll flags, saving custom match sessions for future reuse.
• JS & Math Sandbox: Execute JavaScript scripts (BigInt arithmetic, complex functions) and basic mathematical expressions in a secure local execution engine with stdout/stderr stream views. Load saved project files directly into the console to run them.
• Documentation Vault: Fast offline reference specs for JSON rules, Regex anchors & assertions, Common Shell/Bash navigation commands, HTML/Markdown layout options, and HTTP status codes.

Brutalist Dark Aesthetic:
Featuring a custom industrial high-contrast dark theme optimized for OLED screens, DevPocket feels like a premium terminal designed for deep developer productivity.

100% Privacy & Offline-First:
All code files, regex sessions, and scripts are stored securely on your device inside an offline Room database. No telemetry, no cloud sync, and no internet required. Your source code stays on your device.
